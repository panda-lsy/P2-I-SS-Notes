# Fibonacci Sequence

斐波那契数列是一个自然数序列，定义如下：

* $F_1=1$;
* $F_2=1$; 和
* $F_n=F_{n−1}+F_{n−2} $    $for $   $    n>2$.

编写一个程序输出斐波那契数列中的前 **5** 个数字。

### 输入

此题没有输入。

### 输出

输出 **5** 个整数，表示斐波那契数列中的前 **5** 个数字。输出中的任意两个相邻数字之间恰好用一个空格分隔，行末没有多余的空格或符号。

### 示例 1


| Input      | Output    |
| ---------- | --------- |
| (no input) | 1 1 2 3 5 |
